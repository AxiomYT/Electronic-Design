# DHCP - Dynamic Home Computational Probe

* Arduino UNO Shield
* I2C Communication
* TX + RX + PWR LED's
* 10 Pin IDC RS232 Header
* 3 Pin Temp. Sensor Header
* Vᵢₙ 7 - 12v or USB Bus Power
* 4x Power Rails for Ultrasonic Sensor
* 4x ECHO + TRIG pin passthrough to Raspberry PI
